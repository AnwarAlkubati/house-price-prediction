# House Price Prediction App

Machine learning web application that predicts house prices using housing features.

## Live Demo

https://house-price-prediction-crzjr2o3qgwuwnuhqerkgo.streamlit.app/

## Features

* Predict house prices instantly
* Interactive Streamlit web interface
* Trained machine learning model
* Public deployment online

## Tech Stack

* Python
* Pandas
* NumPy
* scikit-learn
* Streamlit

## Project Structure

* `app.py` → Streamlit application
* `models/model.pkl` → trained model
* `notebooks/house_price_prediction.ipynb` → full notebook
